Что бы запустить проект 
1:создать venv python -m venv venv                  или python -m venv .venv
2:активировать venv .venv\Scripts\activate
3:pip install -r requirements.txt
4:python main.py
--------------------------------------------------------------------------------------------------
Чтобы создать продукт
отправить POST запрос на URL- http://localhost:5000/api/v1/product
{
"id": "на выбор строка",
"price": на выбор число,
"name":"на выбор строка"
}
если id уже занят - продукт обновиться о чём будет свидетельствовать надпись "Продукт был обновлён"
--------------------------------------------------------------------------------------------------
Чтобы получить продукт по id
отправитьGET запрос на URL- http://localhost:5000/api/v1/product/(Здесь следует указать id)
--------------------------------------------------------------------------------------------------
Чтобы получить список продуктов 
отправить GET запрос на URL- http://localhost:5000/api/v1/product?page=0&limit=10
